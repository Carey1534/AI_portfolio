# AI Automation Portfolio

Production AI and automation systems built for real business environments. This repo serves as an overview — individual workflow collections are broken out into dedicated repos below.

---

## Repositories

### [n8n-ai-engine](https://github.com/Carey1534/n8n-ai-engine)
A modular AI reasoning system built progressively over several weeks. Covers deterministic LLM execution, RAG pipelines with pgvector, intent routing, rule proposal generation, and ML-gated policy decisions.

**Stack:** n8n · OpenAI · Supabase pgvector · PostgreSQL

---

### [n8n-client-automations](https://github.com/Carey1534/n8n-client-automations)
Plug-and-play automation workflows for small business operations — appointment reminders, lead response, expense categorization, inventory alerts, contract monitoring, and more.

**Stack:** n8n · OpenAI · Gmail · Google Sheets · Slack · Twilio · Supabase

---

### [n8n-personal-tools](https://github.com/Carey1534/n8n-personal-tools)
Personal productivity and AI-assisted automation workflows including resume generation, freelance market research, emotional check-ins, document OCR pipelines, and client intake systems.

**Stack:** n8n · OpenAI · AWS (S3, Textract) · Supabase · Gmail · Telegram

---

### [n8n-sbb-workflows](https://github.com/Carey1534/n8n-sbb-workflows)
End-to-end client pipeline for a real business — lead intake from Gmail and Tally forms, normalization, AI-drafted responses, and scheduled Slack analytics reports.

**Stack:** n8n · OpenAI · Google Sheets · Gmail · Slack

---

### [expense-saas-pipeline](https://github.com/Carey1534/expense-saas-pipeline)
Production-grade OCR receipt ingestion and expense management platform for multi-tenant financial SaaS. Built with multi-tenant isolation, confidence gating, audit trails, and a full Next.js frontend.

**Stack:** TypeScript · Next.js · AWS Textract · S3 · Supabase · OpenAI · n8n
